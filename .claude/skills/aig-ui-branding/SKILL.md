# AIG Web App Style Guide

This is a style guide used in the building and maintaining Angular and React Applications. It instructs how to layout the top level UI and apply AIG corporate look and feel.

## Background and Font Colours

Header Background Colour: #001871
Header Foreground Colour: White

Content Background Colour: White
Content Foreground Colour: #343741

## Layout

Please use the [AIG LOGO](assets/aig-logo.svg) and copy this to the relevant public assets folder in the relevant Angular or React app so that it can be referenced within the HTML or JSX files of the project.

Generally, the application should be responsively designed. The main header should be the full width of the page, 5rem height and follow the header background and foregound colour guide above.

The content container should be centered vertically about 80% of page width. The header should include the AIG LOGO aligned in the header to the start of the content container and should have a white colour.
